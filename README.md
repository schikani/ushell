# ushell
### ushell is simple yet powerful Bash like terminal for devices running MicroPython

## Commands
```
pwd, cd, cp, head, cat
touch, ls, whoami, networks,
mv, rm, mkdir, venvs, run,  
mkenv, activate, deactivate, 
wificonnect, wifiadd, clear,
ifconfig, platform, repl,
upip, write, reboot

```
